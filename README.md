# InstaHash

This app is built using React Native.
It is React Native Example application.

### Installation

Clone repository:
```sh
git clone git@github.com:edoko/react-native-example-instahash.git
```
Before running:
```sh
npm install
```

## Run

### Android

For using Android OS:

```sh
react-native run-android
```
